## 🙋‍♀️ 스터디 진행 방식

- 시간 : **매주 토요일 큐시즘 세션 전**
- 장소 : **큐시즘 세션 장소 근처**
- 공부 방식 : **스프링 프레임워크 커리큘럼에 querydsl을 녹여 스터디를 진행. 2부로 Golang 언어에 대해 스터디를 진행  / 참고 자료 : Tucker의 Go 언어 프로그래밍**
- 설명 방식 : **Chaper별 정리되어 있는 목차 순서대로 내용 정리 / 실습 코드 리뷰**
- 기록 방식 : **본 레포지토리에 PR로 기록**
- 인증 방식 : **PR 리뷰**
